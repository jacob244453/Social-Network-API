  ## Installation
  1. ```$ git clone```
  2. ```$ cd <this repo>``` 
  3. ```$ npm install``` 
  4. ```$ mongod``` 
  5. ```$ mongo``` 
  6. Open Insomnia Core or your choice of API route testing software and query the database.
